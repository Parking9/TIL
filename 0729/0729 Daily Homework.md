## 2번. Magic Method

- `__init__` : 인스턴스가 생성 될 때 적용하는 메서드
- `__del__`   : 인스턴스가 지워질 때 
- `__str__`   : 인스턴스를 print로 출력할 때 
- `__repr__` : 인스턴스 자체가 반환될 때



## 3번. Instance Method

- `append()` - 리스트의 인자 추가
- `lower()`, `upper()` - string의 소문자, 대문자 반환
- `index()` - 리스트의 대상 인덱스(순서) 출력
