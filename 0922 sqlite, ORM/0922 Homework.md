## 3번. RDBMS, NOSQL

RDBMS는 관계형 데이터 모델로, 모든 데이터를 2차원 테이블 형태로 표현해주는 데이터 모델이다.

NOSQL은 데이터간의 관계를 정의하지 않는다. 분산형 구조로, 데이터를 여러 대의 서버에 분산해 저장. 복잡도와 용량 한계를 극복하기 위해 만들어진 만큼 대용량의 데이터를 저장할 수 있다. Schema가 없어서 데이터에 대한 규격화된 결과 값을 얻기 힘들다.
