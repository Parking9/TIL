## 4번. Static web and Dynamic web

- Static web
  - 미리 저장된 파일이 그대로 전달됨. 고정된 웹 페이지
  - 서버 내에 미리 만들어둔 HTML 페이지를 그대로 사용자에게 전달해준다.
- Dynamic web
  - 사용자의 요청에 따라 데이터가 가공처리한 후 생성되어 전달됨.
  - 사용자 요청에 따라 웹 프레임워크(Django)를 통해 상황에 맞게 페이지를 만들어 사용자에게 제공.
