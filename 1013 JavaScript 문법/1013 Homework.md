## 2번.

- map - 배열의 모든 요소에 특정한 함수 적용
- filter - 배열에서 조건에 맞는 요소만 선택
- find - 배열에서 조건에 맞는 요소의 첫번째 인덱스 도출
- every - 배열에서 주어진 판별 함수를 모두 통과하는지 하나라도 걸리는지 확인하는 함수
- some - 배열에서 주어진 판별 함수를 하나라도 통과하는지 모두 통과 못하는지 판별
- reduce - 배열의 숫자 총합, 평균 계산 등 하나로 줄이는 동작
